== Description ==

This plugin will collect Google Page Rank and Alexa Rank and display it in the footer of your blog.
The result will be collected 1 time each day.

*This release is compatible with all WordPress versions since 3.0.*

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. The plugin will automatically collect Google Page Rank and Alexa Rank
4. On the plugin settings page it is possible to change how the footer is displayed

== Frequently Asked Questions ==

= How often is Google Page Rank and Alexa Rank collected ? =

Every 86400 seconds (1 time per day).
